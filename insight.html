Why creating the insight calculator ?

I'm playing an online tactical game called "dofus" with this kind of mecanic and I wanted to code it myself to understand it better

How does it work ?

It makes a segment between the position of the player and the wanted cell, and then it checks if the segment a wall. If so, it means the player can't see the cell.

I used TKinter for the interface
