What is Logan?

My friends and I have been hosting multiple bots on a common server we have, to help us interract with all of them, I created "Logan", a single bot coded in python acting as a management and interaction tool for other bots(Coded in JS and Python) running on a server.

How does it work ?

Logan has a list discordID who are basically admins who can use him, he mainly uses os.system python function to execute shell commands. He also uses screen sessions to help him manage bots

For example, to restart a bot, Logan does:

  • Search for a screen session with the bot-name as a name
  • Kills the sessions
  • Create a new session with the bot-name as a name with the order to update the bot repo and rerun it
